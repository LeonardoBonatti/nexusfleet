1. Visão Geral do Projeto

O NexusFleet é um painel logístico (Dashboard) corporativo desenvolvido para automatizar, monitorar e otimizar a gestão de frotas, consumos de combustível e rotinas de manutenção preventiva. O sistema foi ideado a partir de uma necessidade acadêmica e mercadológica estruturando uma interface rica e altamente responsiva.

2. Arquitetura e Tecnologias

A construção do software baseia-se em um ecossistema moderno, utilizando os seguintes pilares tecnológicos:

  • Frontend Interativo: HTML5, JavaScript Moderno (ES6+) e CSS3 utilizando conceitos de Glassmorphism, variabilidade de temas (Dark Mode) e componentização arquitetônica.
  • Backend Server: Motor NodeJS rodando via Express.js, padronizado com MVC.
  • Banco de Dados (Nuvem): PostgreSQL Serverless na plataforma Neon.tech, com ORM adaptado e String SSL segura.
  • Módulo Geo-Espacial: Integração com a abstração vetorial LeafletJS focado na roteirização de mapas customizados gratuitos.
  • Progressive Web App (PWA): Arquitetura de Service Workers garantindo velocidade com cache persistente.

3. Fluxo de Autenticação e Segurança Estratégica

O sistema atua com uma camada robusta e blindada de acesso inicial:

  • Autenticação dividida em dois fatores visuais (Credencial corporativa seguida por módulo "Vault" de validação numérica tipo MFA).
  • Armazenamento de rotas criptografadas e restrição de rotas não autorizadas utilizando "Auth Guards", barrando acessos ao app.js sem o token válido na memória.
  • Hospedagem dinâmica via túneis seguros (ngrok/localtunnel) para apresentação cross-device (Sincronização Nuvem PC + Celular).

4. Componentes Operacionais

O framework do NexusFleet fornece os seguintes subsistemas para o controle autônomo da transportadora:

  • Painel de Operações (Dashboard): Agregador de KPIs e monitoramento térmico de despesas (verdes x vermelhos) calculando o Delta percentual do mês;
  • Controle de Veículos e Abastecimento: Inputs formulados em tempo real de placa, Hodômetro e combustível injetado, estruturados para o cálculo de despesas;
  • Modais de Perfil: Sistema multilinguagem capaz de varrer nós neurais para tradução via APIs de terceiros.

5. Conclusão Acadêmica

O presente projeto documenta com clareza todo o esforço de engenharia de software da dupla avaliada. O **NexusFleet** não apenas resolve a necessidade de gestão da frota, mas propõe uma interface corporativa de altíssimo nível alinhada aos modernos conceitos de Web Apps.
